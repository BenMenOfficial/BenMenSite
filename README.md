## ✨ תכונות

### 🎨 עיצוב וחוויית משתמש
- **עיצוב מודרני ומינימליסטי** עם פלטת צבעים מקצועית
- **רספונסיבי מלא** - מותאם לכל המכשירים (מובייל, טאבלט, דסקטופ)
- **אפקטים ויזואליים מתקדמים**:
  - Digital Rain Effect - אפקט גשם דיגיטלי ברקע
  - Confetti Effect - אפקט קונפטי בעת שליחת טופס
  - אנימציות חלקות וטרנזישנים
- **תפריט ניווט נפתח** למובייל עם תמיכה מלאה ב-ARIA
- **גלריית עבודות אינטראקטיבית** עם Lightbox ותמיכה במחוות מגע

### 📱 פונקציונליות
- **טופס יצירת קשר** עם אינטגרציה ל-Formspree
- **מדיניות פרטיות** מובנית עם Lightbox
- **סקשן FAQ** עם אקורדיון אינטראקטיבי
- **מונה מבקרים** מבוסס LocalStorage
- **סטטיסטיקות אנימטיביות** עם אפקט ספירה
- **כפתור חזרה למעלה** עם אנימציה חלקה

### 🔍 SEO ואופטימיזציה
- **מטא-תגים מלאים** (Open Graph, Twitter Cards, WhatsApp)
- **Schema.org Structured Data** מובנה
- **Sitemap.xml** מעודכן
- **Robots.txt** מוגדר
- **Google Analytics (GA4)** מוטמע
- **PWA Support** - Progressive Web App עם Manifest.json
